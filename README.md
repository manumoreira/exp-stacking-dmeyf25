## Intro
Los experimentos colaborativos en el contexto de la cátedra DMEyF 2025 tienen el objetivo de invitar a grupos de alumnos a explorar temas prácticos de data mining para compartirlos con sus compañeros

## Experimento stacking

En este caso hicimos algunos experiemntos poniendo a prueba la técnica de stacking. 

Calculamos predicciones para nuestra clase en distintas ventanas de tiempo y con distintos modelos predictivos y luego los sumamos como features del dataset para luego volver a correr otro modelo predictivo que toma estas preducciones como input
